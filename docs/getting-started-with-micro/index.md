[Snowplow Micro](/docs/testing-debugging/snowplow-micro/what-is-micro/index.md) is a minified version of the Snowplow pipeline that can run on your machine. For a few quick tests, Micro can be a simple alternative to deploying an full-blown Snowplow Open Source pipeline to your cloud.
