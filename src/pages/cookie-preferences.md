## What are cookies?

Cookies are small text files stored on your device when you visit a website. We use cookies to make the website function correctly, keep it secure, and understand how visitors use it.

## How do we use cookies?

We use first-party and third-party cookies for several purposes. The sections below list the main cookies used on this website.

### Essential services

| Cookie               | Description                                                                                   |
| -------------------- | --------------------------------------------------------------------------------------------- |
| `_ketch_consent_v1_` | Stores your explicit cookie preferences after you interact with the consent banner.           |
| `_swb`               | Set by Ketch to identify your browser session for consent management purposes.                |
| `_swb_consent`       | Stores your current consent state, including the legal basis applied to each cookie category. |

### Data analytics

| Cookie                        | Description                                                                                                                                                                                               |
| ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `_ga`                         | Installed by Google Analytics. Calculates visitor, session, and campaign data and tracks site usage. Stores information anonymously and assigns a randomly generated number to recognize unique visitors. |
| `_gid`                        | Installed by Google Analytics. Stores information on how visitors use the website and generates an analytics report of site performance.                                                                  |
| `_gat_gtag_UA_159566509_1`    | Set by Google to distinguish users.                                                                                                                                                                       |
| `_sp5_id` and `_sp_biz1_id`   | Set by Snowplow. Store a unique identifier for each user, a unique identifier for the current session, the number of visits, and timestamps for the first visit, previous visit, and current visit.       |
| `_sp5_ses` and `_sp_biz1_ses` | Set by Snowplow. Used to identify whether a user is in an active session or whether this is a new session.                                                                                                |
| `sp`                          | Stores a Snowplow server-generated unique identifier sent with all subsequent tracking events for the same user.                                                                                          |

## How can I control the cookie preferences?

Click the **Preferences** button above, or the cookies widget in the bottom corner of the page, to enable or disable cookie categories at any time.

You can also control cookies through your browser settings. To learn more about managing and deleting cookies, visit [allaboutcookies.org](https://www.allaboutcookies.org).
