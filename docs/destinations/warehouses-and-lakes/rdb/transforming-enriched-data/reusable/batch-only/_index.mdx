    <td><code>input</code></td>
    <td>Required. S3 URI of the enriched archive. It must be populated separately with <code>run=YYYY-MM-DD-hh-mm-ss</code> directories.</td>
</tr>
<tr>
    <td><code>runInterval.*</code></td>
    <td>Specifies interval to process.</td>
</tr>
<tr>
    <td><code>runInterval.sinceAge</code></td>
    <td>Optional. A duration that specifies the maximum age of folders that should get processed. If <code>sinceAge</code> and <code>sinceTimestamp</code> are both specified, then the latest value of the two determines the earliest folder that will be processed.</td>
</tr>
<tr>
    <td><code>runInterval.until</code></td>
    <td>Optional. Process until this timestamp.</td>
</tr>
<tr>
    <td><code>monitoring.sentry.dsn</code></td>
    <td>Optional. For tracking runtime exceptions.</td>
</tr>
<tr>
    <td><code>deduplication.*</code></td>
    <td>Configure the way in-batch deduplication is performed</td>
</tr>
<tr>
    <td><code>deduplication.synthetic.type</code></td>
    <td>Optional. The default is <code>BROADCAST</code>. Can be <code>NONE</code> (disable), <code>BROADCAST</code> (default) and <code>JOIN</code> (different low-level implementations).</td>
</tr>
<tr>
    <td><code>deduplication.synthetic.cardinality</code></td>
    <td>Optional. The default is 1. Do not deduplicate pairs with less-or-equal cardinality.</td>
</tr>
<tr>
    <td><code>deduplication.natural</code></td>
    <td>Optional. The default is 'true'. Enable or disable natural deduplication. Available since <code>5.1.0</code></td>
</tr>
