
</td>

    <td>Configure the way in-batch deduplication is performed</td>
</tr>

    <td><code>deduplication.synthetic.type</code></td>
    <td>Optional. The default is <code>BROADCAST</code>. Can be <code>NONE</code> (disable), <code>BROADCAST</code> (default) and <code>JOIN</code> (different low-level implementations).</td>

<tr>
    <td><code>deduplication.synthetic.cardinality</code></td>
    <td>Optional. The default is 1. Do not deduplicate pairs with less-or-equal cardinality.</td>
</tr>
<tr>

    <td>Optional. The default is 'true'. Enable or disable natural deduplication. Available since <code>5.1.0</code></td>

</tr>
<tr>
    <td><code>skipSchemas</code> (since 5.7.1) </td>
    <td>Optional, default = none. Supply a list of Iglu URIs and the transformer's output files will omit any columns using that schema.  This feature could be helpful when recovering from edge-case schemas which for some reason cannot be loaded to the table.</td>
</tr>
