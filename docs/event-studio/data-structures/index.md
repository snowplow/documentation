* Snowplow Console UI
* Data structures API
* [Snowplow CLI](/docs/event-studio/programmatic-management/snowplow-cli/index.md)
* For Community users: [Iglu](/docs/api-reference/iglu/iglu-repositories/iglu-server/index.md)
